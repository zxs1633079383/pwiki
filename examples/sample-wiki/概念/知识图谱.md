# Knowledge Graph (knowledge-graph)

> A unified graph model that lets an LLM navigate code, docs, and decisions as one structure.

## Core idea

Nodes are concepts, entities, and operations. Edges are typed relationships.
The LLM treats this graph as a programming environment: it reads what's there,
asks questions, writes back conclusions as new nodes.
